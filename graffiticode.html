Graffiti Code

Graffiti Code is the practice of creating little languages for describing the behavior of expressive computers.

Imagine, for example, a robot that understands time and knows how to move forward and backward and turn right and left by any angle. The following program might be used to describe a behavior of that robot.

on dance n. if n is zero then stop. move forward two seconds. turn right ninety degrees. move backward two seconds. turn right ninety degrees. move forward two seconds. turn left ninety degrees. move forward two seconds. turn left ninety degrees. do dance n minus one. no dance.

do dance ten.

This program defines a recursive function 'dance' that terminates when its parameter 'n' is equal to 'zero'. The 'on' keyword introduces a function definition. The 'do' keyword introduces an invocation of a function. The 'no' keyword marks the end of a function definition. The 'move' and 'turn' commands describe behaviors that have platform specific implementations. The 'minus' operator is a builtin arithmetic operator.

You've probably noticed that the only punctuation used is the period, and that numbers are spelled out as words. These features allow this program to be typed into a smart phone keyboard without switching modes. For this same reason the language is case insensitive and whitespace is only significant for separating tokens.
