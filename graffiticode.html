Graffiti Code

Graffiti Code is the practice of creating specialized environments to enable people to express themselves with code. We eliminate complexity from the programming process by focusing tools and vocabularies on a specific task domain. We also demystify the coding experience by giving users real-time feedback as they edit their code.

One application of Graffiti Code can be found at graffiticode.org. This web application allows users to make web comptabiel drawings with code. Drawings begin with other drawings and are often remixed and shared on the web.

Commissions and Collaborations. We seek commissions for Graffiti Code works, as well as collaborations that improve this practice. The following are a few such projects we have in mind:

  • Working with educational technology providers to create languages and environments for programming physical computers.
  • Working with designers and artists to invent little languages for describing the shape and behavior of expressive media.
