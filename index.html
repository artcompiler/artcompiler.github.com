Art Compiler does work in the space between art and code. Our purpose is to use language to create visual works of practical and aesthetic value. Our practice is built on technical expertise acquired through decades of language design and compiler development and is driven by the desire to use computing as a platform for expressing meaning in the world.

Language is our medium. At the core of our work are specialized compilers that translate language into meaningful objects and behaviors. Each language defines a mapping between the domain of code and the domain of experience. It is precisely this mapping that gives meaning to that code and which we care about most.

We are currently engaged in practices we call Code Cartography and Graffiti Code. In Code Cartography we map program behavior onto syntactic representations of code to allow programmers to see more clearly hidden conequences. In Graffiti Code we invent specific languages for describing the behavior of expressive computers.
